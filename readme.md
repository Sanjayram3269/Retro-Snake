# Retro Snake – AI Coach

A classic Snake game revived with a modern AI twist.

## Features
- Retro grid-based Snake gameplay
- Real-time AI coach that analyzes board state
- Visual AI move suggestion (ghost cell)
- Risk labels: SAFE / RISKY / DEADLY

## AI Logic
The AI evaluates all legal moves each frame using heuristics based on
distance to food and collision avoidance, providing explainable guidance.

## Tech
- JavaScript (ES Modules)
- HTML Canvas
- Rule-based AI heuristics
